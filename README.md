## 简介

valuka-admin 是一个免费开源的中后台模版。使用了最新的`vue3`,`vite`,`TypeScript`等主流技术开发，开箱即用的中后台前端解决方案。

## 特性

- **最新技术栈**：使用 Vue3/vite 等前端前沿技术开发
- **TypeScript**: 应用程序级 JavaScript 的语言
- **权限** 内置完善的动态路由权限生成方案
- **组件** 二次封装了多个常用的组件

## 安装使用

- 获取项目代码

```bash
git clone https://github.com/ValukaOrganization/valuka-admin.git
```

- 安装依赖

```bash
cd valuka-admin
pnpm install
```

- 运行

```bash
pnpm dev
```

- 打包

```bash
pnpm build
```
